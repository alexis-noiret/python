Ensemble de scripts Python organisés en exercices progressifs visant à développer une base solide en **programmation**, **algorithmique** et **scripting**.

---

## Résumé exécutif

Développement d’une série d’exercices Python permettant de maîtriser les fondamentaux du langage et de construire une base exploitable pour des usages avancés :

* automatisation système
* scripting en cybersécurité
* développement d’outils techniques

---

## Périmètre technique

* Langage : Python 3
* Exécution : ligne de commande
* Organisation : scripts indépendants
* Approche : apprentissage progressif

---

## Réalisations clés

* Développement de multiples scripts Python (job01 → job15)
* Mise en pratique des structures fondamentales du langage
* Structuration logique du code
* Résolution de problèmes algorithmiques
* Exécution et validation en environnement Linux

---

## Architecture du projet

```text
python/
└── jour6_python/
    ├── job01.py
    ├── job02.py
    ├── job03.py
    ├── ...
    ├── job15.py
    └── main.py
```

---

## Fonctionnalités

Selon les exercices réalisés :

* manipulation de variables et types
* structures conditionnelles
* boucles (for / while)
* fonctions
* traitement de données
* scripts interactifs

---

## Exécution

Lancer un script :

```bash
python3 job01.py
```

Exécuter un autre exercice :

```bash
python3 job10.py
```

Ou via un point d’entrée global :

```bash
python3 main.py
```
